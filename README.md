# Project Phoenix — 2026 Doctrine

Project Phoenix is a sovereign digital and physical infrastructure system for 2026.
It delivers agentic AI, deterministic execution, mapless autonomy, secure identity,
and autonomous manufacturing to replace legacy systems with resilient,
energy-efficient, always-on intelligence.

## Purpose

This repository defines the locked strategic doctrine for Project Phoenix.
It establishes market timing, sovereign positioning, and the minimum viable
deployment scope for 2026.

All Phoenix sub-projects reference this repository as their authoritative source.

This repository contains no experimental code.
It defines what is allowed to exist.
